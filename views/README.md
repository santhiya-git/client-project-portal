# Client Project Portal

## Project Description

Client Project Portal is a web-based project management application developed using Node.js, Express.js, MySQL, HTML, CSS, JavaScript, and EJS. It helps organizations manage client projects by allowing users to add, view, and monitor project details through an interactive dashboard.

---

## Features

* User Login
* Dashboard
* Add New Project
* View All Projects
* MySQL Database Integration
* Dynamic Project Management
* Professional Responsive UI

---

## How to Run

1. Install Node.js
2. Install dependencies

```
npm install
```

3. Start MySQL

4. Run the server

```
node server.js
```

5. Open

```
http://localhost:5000
```
